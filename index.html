Example usage

Resolv using default settings

node /box/ {
  include resolv
}

Resolv using custom settings

node /box/ {
  class { 'resolv':
    resolv_search      => $::fqdn,
    resolv_domain      => $::domain,
    resolv_nameservers => [ '208.67.220.220', '208.67.222.222' ]
}
